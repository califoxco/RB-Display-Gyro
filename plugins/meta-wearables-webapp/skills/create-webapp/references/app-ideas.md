# App Ideas for Meta Display Glasses

Suggest from these categories when the user hasn't specified what to build.

## Data-Connected Apps (use real APIs)

| App | API Source | Features |
|-----|-----------|----------|
| Weather Dashboard | Open-Meteo API | Current conditions, forecast, location |
| News Reader | NewsAPI / RSS | Headlines, categories, article view |
| Crypto Tracker | CoinGecko API | Live prices, charts, watchlist |
| Stock Ticker | Alpha Vantage / Yahoo | Prices, portfolio, alerts |
| Wikipedia Explorer | Wikipedia API | Search, article view, random |
| Trivia Live | Open Trivia DB API | Categories, difficulty, scoring |
| Recipe Finder | TheMealDB API | Search, ingredients, steps |
| Space Tracker | NASA API | APOD, Mars photos, ISS position |
| GitHub Status | GitHub API | Repo stats, contribution graph |
| Air Quality | OpenAQ API | AQI, pollutants, city comparison |
| Earthquake Monitor | USGS API | Recent quakes, map, magnitude |
| Dictionary | Free Dictionary API | Word lookup, pronunciation, examples |
| Movie Browser | OMDb API | Search, details, ratings, watchlist |
| IP Geolocation | ipapi.co | Location, ISP, timezone info |
| Random Facts | Various APIs | Cat facts, jokes, numbers, quotes |
